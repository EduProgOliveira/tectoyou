# tectoyou

Projeto faculdade TCC

Criação de duas plataformas APP e DESKTOP liguagem JAVA.

Projeto visa em uma aplicativo mobile onde o cliente irar (SE CADASTRAR, DEPOSITAR, SACAR, "COMPRAR E VENDER BITCOINS", E FAZER INVESTIMENTO)

Os investimentos são feitos em empresas cadastradas e elas terão acessa aos seus investidores "CLIENTES" pela plataforma DESKTOP.

Lembrando que será fictício para fins academicos.
